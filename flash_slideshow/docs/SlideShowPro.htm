SlideShowPro

The complete photo gallery solution for Flash

SlideShowPro is a component for Macromedia Flash MX 2004 and Flash 8 that helps you display pictures on your web site. With over 60 customizable options, SlideShowPro works how you want it to, and can easily be styled and resized to match your existing site design — all with a simple point and click user interface.

Thousands of web sites — from personal to high-traffic destinations like Rolling Stone, ESPN and The New York Times — use SlideShowPro to display photography, editorial slide shows, design portfolios, and countless other creative uses.

Check out the full feature set of SlideShowPro, or view a screencast to see how it all works.

Your browser does not have Flash Player 8. Download Flash Player 8.

SlideShowPro Director

Ajax powered photo administration for SlideShowPro

Say goodbye to coding XML and generating thumbnail images by hand. SlideShowPro Director is built from the ground-up to fully administer one or more slide shows created with SlideShowPro. Completely browser based and chocked-full of Ajax interactivity, Director is the quickest way to get your images into SlideShowPro. It generates thumbnails, provides image captioning inputs, manages albums, sorts images, publishes XML data, and generates dynamic galleries.

And because Director was built for SlideShowPro, you can count on it to update right alongside the Flash component whenever new features are introduced.

Check out the full feature set of Director for more information, or view a highlight video for a quick overview.
