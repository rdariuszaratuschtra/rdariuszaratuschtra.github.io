Create htaccess File

.htaccess / .htpasswd File Generator

You need to create an .htaccess and an .htpasswd file to protect a special directory in your site. This tool will create htaccess and htpasswd file automatically. Use them to password protect a directory of oyur website. Simply enter the username and password that you require and the exact path of the directory you want to protect.

 

Username:
   The username you want to use to access the protected directory.

Password:
   Choose a password. It will be encoded with the strong MD5 encryption.

Path:
   Path to your protected directory from your site's root. No trailing slash! (e.g.:"/htdocs/html/secret")
