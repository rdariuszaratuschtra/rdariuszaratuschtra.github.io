Transitions and Filters

The menu at left demonstrates the effects of applying IE transitions and filters to HM Menus. The first menu item at left pops up the child menu in its unfiltered, untransitioned state. The remaining menu items pop up child menus with the specified filter/transition applied.

Only one "show" and "Hide" transition can be supplied for each HM menu; but multiple "static" filters can be applied to a menu. The examples here only include one filter per menu, but you could combine two or more to create a cumulative effect. For example, a menu could have both a Shadow and Glow applied to it.

For a complete listing of possible IE transitions and filters, see the MSDN Reference:

MSDN: Visual Filters and Transitions Reference

Also see the HM documentation for the following properties:

IEShowTransition
IEHideTransition
IEFilters
